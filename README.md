# 个人密码本文档系统 python1133

## 项目概述

个人密码本文档系统是一个帮助用户安全、便捷地管理其账户密码的工具。它采用Python语言开发，结合Django框架和MYSQL数据库，提供密码创建、管理、强度检测、自动输入等一站式密码管理服务。

## 技术栈

- Python
- Django
- MYSQL

## 功能模块

### 新建密码本

用户可以创建新的密码本来记录账户和密码。

### 平台名称备忘


### 密码强度检测

用户可以查看所设密码的安全强度，并据此决定是否需要修改为更安全的密码。

### 密码表生成

用户可以从系统生成的密码表中选择任意密码保存使用。

### 自动输入

在使用密码本中保存的平台登录时，用户可以执行账户密码的自动输入。

### 密码本管理

用户可以随时对密码本进行管理，包括删除、修改和停用等操作。

### 导出功能

用户可以将任意密码本导出为.txt格式文件保存。

## 系统角色

- 用户：使用系统的主体，可以创建和管理密码本。
